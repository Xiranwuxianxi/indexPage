
# 🌌 Xiran IndexPage

一个基于 Vue3 + Vite 构建的个人数字主页。
不仅仅是展示页面，更像是一个持续进化中的数字实验场。

这里记录我的项目、技术方向、学习理念，以及我对于「代码如何重构现实世界」的思考。

## 🚀 本地运行

克隆项目：

```bash
git clone https://github.com/Xiranwuxianxi/indexPage.git
```

进入项目目录：

```bash
cd indexPage
```

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

---

## 📦 生产构建

```bash
npm run build
```

## 🎨 设计理念

这个项目并不追求复杂功能堆砌。

我更希望它像一个：

> “拥有技术灵魂的个人空间”

因此你会看到：

* 柔和但克制的动态效果
* 大量留白
* 轻科技感视觉
* 模块化信息表达
* 对阅读体验的优化

相比传统个人主页，我更希望它像：

* 一个数字名片
* 一个技术人格入口
* 一个持续成长中的开放实验室
